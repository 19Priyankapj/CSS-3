CSS FLEX

The flexbox or flexible box model in css is a one dimensional layout model that has flexible and efficient layouts with distributed spaces.

flexible box has layout module, makes it easier to design flexible responsive layout structure without using float or positioning.

properties for the parent(flex container)

1) display: flex;

1) flex-direction: row;

3) flex-wrap: wrap;

4) flex-flow: row wrap; ( flex-flow = flex-direction + flex-wrap;)

5) justify-content: center;

6)align-items: center;

7) align-content: space-between;

8) column-gap: 20px;

9) row-gap: 20px;

10) gap: 20px;

properties for the children (flex items)

1) flex: 1; OR flex-grow

2) flex-shrink: 3;

3) order: 2;

4) align-self: center;
